YOUR BASIC INFORMATION

Forms web pages pe user input lene ke liye use hote hain. Forms mein input fields, checkboxes, radio buttons, submit buttons waghera hote hain.

<form> tag se form shuru hota hai. Isme do important attributes hote hain:

  • action: Yeh specify karta hai ke form data kahan send hoga.
  • method: Yeh specify karta hai ke form data kaise send hoga (GET ya POST).
